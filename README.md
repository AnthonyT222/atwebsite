# AT Software Engineering Learning Website

A personal learning journey website documenting progress in software engineering and web development technologies.

## 📋 Project Overview

This website serves as a portfolio and documentation of my software engineering learning journey, covering various programming languages, frameworks, and middleware technologies. The site is built with HTML and CSS, featuring a clean design with breadcrumb navigation and interactive button elements for enhanced user experience.

## 🚀 Features

- **Personal Learning Portfolio**: Showcases learning goals and progress
- **Breadcrumb Navigation**: Easy navigation across all middleware topics
- **Interactive Buttons**: Enhanced button styling with hover and active states
- **Educational Content**: Detailed pages covering middleware concepts
- **Responsive Design**: Background images and flexible layouts
- **External Resources**: Links to learning platforms like Codecademy

## 📁 Project Structure

